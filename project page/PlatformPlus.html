Project Description:

The primary goal of this project is to make a good platformer with fairly decent physics and other game implementations( e.g., UI/HUD, Game Management, Game Design, etc.)

The following phases in this project involve:
  • Developing prototypes to showcase various mechanics implemented.
  • Contributing to the Design and logic of the game( which brings together the development of the game).
  • Non-technical design, such as level design, distinct game modes, subtle narratives, etc. are most welcome as well.
  • Polishing the game after finalizing the prototypes through rendering, Scene Management, graphics management, post-processing, and so on.
  • Creating functionalities for the UI, sound design, and other facets of the game like 3D-modelling, VFX, and animations, you are also free to use open-source assets (although there is a limitation). All the third-party assets to be used must be checked by the maintainer before using them.
  • Looking into optimizations and trying to expand the deployment platforms (e.g., mobile, webGL builds etc.)
