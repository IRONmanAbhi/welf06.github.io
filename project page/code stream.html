Project Description:


Why create a code collaboration tool? :
  • During the pandemic, we all were homebound. Every developer/programmer encounters a bug in their code, which can be solved fast if collaborated with other developers. Codestream is an easy way for developers to share their code with other developers with real ease. Share the link with other developers to get them in the same room and find and fix the bug easily!

  • As an undergrad student going to sit for internships/placement interviews, it’s essential to give mock interviews to prepare. Codestream can be used here to code, collaborate, and run the code to check if it is executable or not.

  • A SaaS tool to connect homebound developers, friends and help them collaborate over issues in an easier way!

Bottlenecks:
  • Codestream scales very well regarding how many rooms can be on the server, but there is a limitation on how many participants can be in one single room.

  • YJS server allows clients to send and receive updates related to shared editor through an intermediary server. We are currently using a publicly available server, but they don’t ensure the speed / reliabilty of the connection and updates, for that we need to deploy a private server.

  • The application’s backend won’t scale (As we are using a free dyno on Heroku :p)

Future Goals:
